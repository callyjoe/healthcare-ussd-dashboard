Healthcare USSD Simulator & Admin Dashboard

Overview
This project is a front-end web application that bridges basic mobile infrastructure with a modern administrative interface. It features a fully interactive USSD mobile phone simulator where users can navigate text-based menus to book clinic appointments or log health symptoms.

As the user interacts with the simulated phone, a companion admin dashboard dynamically updates in real-time, logging the incoming data into an organized activity table.

Key Features & Technical Skills Demonstrated:

    State Management: Uses Vanilla JavaScript to track the user's current position within a multi-step USSD menu flow.

    Real-Time DOM Manipulation: Captures user input and instantly generates new HTML elements (table rows) on the dashboard without reloading the page.

    Responsive Layout Design: Utilizes modern CSS (Flexbox/Grid) to create a clean, split-pane interface that contrasts a retro mobile screen with a modern SaaS dashboard.

    Tech Stack: HTML5, CSS3, Vanilla JavaScript.
